# Boomerang Bandits Plugin

Official RuneLite plugin for **Boomerang Bandits** clan members. Track your clan points, compete on leaderboards, stay on top of clan events, and coordinate with the clan — all from inside the game.

## What it does

- View your clan points, rank, and recent activity in the Profile panel
- Clan leaderboard ranked by total points
- Browse active and upcoming WOM competitions with live leaderboards
- View and create clan events — location, world, password, and countdown all in one place
- In-game overlay shows the active event details while you play
- Clan Hub with quick links, member roster, Dink webhook setup, and points guide
- Coffer deposit sound — plays when a clan member deposits into the clan coffer
- Admin tools for staff: attendance tracking, rank changes, announcements, and roster sync

## Requirements

- Membership in the Boomerang Bandits clan
